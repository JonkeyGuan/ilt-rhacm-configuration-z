This folder holds applications that need per-cluster configuration.

Directory structure should be like this:

.
├── application-name-1
│   ├── base
│   └── overlays
│       ├── cluster-name-1
│       ├── cluster-name-2
│       └── ...
├── application-name-2
    ├── base
    └── overlays
        ├── cluster-name-1
        ├── cluster-name-2
        └── ...
